# Skygen Discord Bot — Privacy Policy

_Last updated: March 2026_

This Privacy Policy explains what data Skygen collects, how it is used, and how you can request deletion.

---

## 1. Information We Collect
Skygen collects only the data required to operate the bot and server.

### 1.1 Discord Data
- Discord ID  
- Username and discriminator  
- Roles and permissions  
- Messages sent to the bot (commands, verification info, applications)

### 1.2 Minecraft / Xbox Data
- Minecraft username  
- UUID  
- Xbox Gamertag  
- Public Xbox profile data  
- Verification timestamps  

### 1.3 Application Data
If you submit an admin application, we collect:
- Responses to application questions  
- Discord ID  
- Submission date  
- Staff decision  

### 1.4 Server Logs
For security and moderation:
- Join/leave logs  
- Punishment logs  
- Verification attempts  
- Bot error logs  

---

## 2. How We Use Your Data
Data is used for:
- Account verification  
- Assigning roles  
- Linking Discord ↔ Minecraft/Xbox accounts  
- Preventing impersonation  
- Processing staff applications  
- Moderation and security  
- Improving bot functionality  

We **never** sell or share your data with third parties.

---

## 3. Data Retention
We keep data:
- As long as your account is active  
- Until you request deletion  
- Longer if required for moderation or security  

Application data may be deleted after review.

---

## 4. Data Sharing
We may share data only with:
- Skygen staff  
- Discord (via API)  
- Minecraft/Xbox APIs  

We do **not** share data with advertisers or external companies.

---

## 5. Data Security
We use:
- Restricted staff access  
- Secure storage  
- Encrypted communication where possible  
- Automated systems to prevent unauthorized access  

No system is 100% secure, but we take reasonable measures to protect your data.

---

## 6. Your Rights
You may request:
- A copy of your stored data  
- Correction of inaccurate data  
- Deletion of your data  

We may deny deletion requests if:
- The data is required for moderation  
- The request is fraudulent  
- The request would compromise server security
